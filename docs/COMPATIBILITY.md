# Compatibility contract

AVlab reports capabilities honestly.

- U0: exact preservation, versions, restore and opaque access.
- U1: package and dependency inspection.
- U2: interchange understanding.
- U3: native semantic reading.
- U4: safe native change application.
- U5: live host integration.

Every DAW, editor and project type receives U0 support. Deeper levels are declared per capability by adapters; they are not inferred from a filename.
